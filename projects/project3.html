Hey App

Promotions /Image recongnition
Reto

Utilizar la realidad aumentada y la geolocalización para mostrar al usuario final promociones y descuentos que se encuentran alrededor.

Solución

La creación de una aplicación que combina la realidad aumentada, la geolocalización y el reconocimiento de imagen para poder mostrar de forma interactiva las promociones que se encuentran alrededor de cada usuario. Utilizamos un sistema de Data Analysis para mostrar información segmentada al usuario y poder ofrecer promociones que den un valor agregado para utilizar la aplicación.

Resultado

Una de las aplicaciones con mayor éxito en el área de marketing, ya que la empresa recibe buen feedback del video, las promociones y la forma en la que renueva su estrategia de marketing.
